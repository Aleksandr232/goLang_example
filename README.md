# Go — учебные примеры

## Урок 1

**Время:** 1 ч 10 мин  
**Папка:** `1/`

### Что прошли

- Объявление переменных: `var` и короткая запись `:=`
- Базовые типы: `int`, `string`, `float32`, `bool`
- Проверка типа через `reflect.TypeOf`
- Условия `if` / `else` и сравнение значений
- Приведение типов (например, `float32` и `int` нельзя сравнивать без конвертации)
- `byte` и срез `[]byte`, вывод символа через `fmt.Printf("%c", ...)`
- Тип `rune` (символ Unicode)
- Множественное присваивание и обмен значений: `a, b, c = c, a, b`

### Файлы

- `1/main.go` — примеры с урока (часть закомментирована для повторения)

### Что прошли

- Оператор `switch` / `case` / `default`

## Урок 4

**Время:** 1 ч 17 мин  
**Папка:** `4/`

### Что прошли

- Указатели: объявление `*int`, взятие адреса `&num`, разыменование `*p`
- Передача указателя в функцию и изменение значения через указатель (`printMessage(msg *string)`)
- Глобальные переменные и функция `init()` для инициализации при старте пакета
- Массивы фиксированной длины: `[3]string{"hello", "world", "golang"}`
- Слайсы: литерал `[]string{...}`, создание через `make([]string, 3)`, добавление элементов через `append`
- Длина и ёмкость слайса: `len()` и `cap()`
- Двумерные слайсы: литерал `[][]int{...}` и построение матрицы через вложенные `make` и циклы
- Цикл `for range` по слайсу: индекс и значение (`for index, message := range messages`)
- Функция со слайсом и возвратом ошибки: `printMessages(messages []string) error`
- Проверка пустого слайса и возврат ошибки через `errors.New`

- `4/main.go` — примеры с урока (часть закомментирована для повторения)
