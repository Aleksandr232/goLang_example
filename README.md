# Go — учебные примеры

## Урок 1

**Время:** 1 ч 10 мин  
**Папка:** `1/`

### Что прошли

- Объявление переменных: `var` и короткая запись `:=`
- Базовые типы: `int`, `string`, `float32`, `bool`
- Проверка типа через `reflect.TypeOf`
- Условия `if` / `else` и сравнение значений
- Приведение типов (например, `float32` и `int` нельзя сравнивать без конвертации)
- `byte` и срез `[]byte`, вывод символа через `fmt.Printf("%c", ...)`
- Тип `rune` (символ Unicode)
- Множественное присваивание и обмен значений: `a, b, c = c, a, b`

### Файлы

- `1/main.go` — примеры с урока (часть закомментирована для повторения)
