# Go — учебные примеры

## Урок 1

**Время:** 1 ч 10 мин  
**Папка:** `1/`

### Что прошли

- Объявление переменных: `var` и короткая запись `:=`
- Базовые типы: `int`, `string`, `float32`, `bool`
- Проверка типа через `reflect.TypeOf`
- Условия `if` / `else` и сравнение значений
- Приведение типов (например, `float32` и `int` нельзя сравнивать без конвертации)
- `byte` и срез `[]byte`, вывод символа через `fmt.Printf("%c", ...)`
- Тип `rune` (символ Unicode)
- Множественное присваивание и обмен значений: `a, b, c = c, a, b`

### Файлы

- `1/main.go` — примеры с урока (часть закомментирована для повторения)

## Урок 2

**Время:** 43 мин  
**Папка:** `2/`

### Что прошли

- Функции с несколькими возвращаемыми значениями: `(string, bool, error)`
- Цепочка условий `if` / `else if` / `else`
- Создание ошибок через `errors.New`
- Обработка ошибок: проверка `err != nil`
- Логирование и завершение программы через `log.Fatal`
- Пример: функция `enterTheClub` — проверка возраста для входа в клуб

### Файлы

- `2/main.go` — функция `enterTheClub` с возвратом сообщения, флага и ошибки

## Урок 3

**Время:** 40 мин  
**Папка:** `3/`

### Что прошли

- Оператор `switch` / `case` / `default`
- Пакет `time`: автоматическое определение дня недели через `time.Now().Weekday()`
- Функция `prediction` — пожелания на каждый день недели
- Вариативные функции: `findMin(numbers ...int)` — произвольное число аргументов
- Цикл `for range` для перебора слайса
- Анонимные функции и немедленный вызов: `func() { ... }()`
- Замыкания (closures): функция, возвращающая функцию и сохраняющая состояние (`innerFunction` со счётчиком)

### Файлы

- `3/main.go` — примеры с урока (часть закомментирована для повторения)
